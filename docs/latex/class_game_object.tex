\doxysubsection{Detailed Description}
A base class for user-\/defined game objects in the engine. 

\doxylink{class_game_object}{Game\+Object} extends the \doxylink{class_object}{Object} class and represents a typical updatable and drawable entity in the scene. It is intended for general-\/purpose gameplay logic.

This class provides default empty implementations for all lifecycle functions such as Init, Update, and Draw. Users can subclass \doxylink{class_game_object}{Game\+Object} to implement their own logic and behaviors by overriding these methods.

Game\+Objects are typically registered through the \doxylink{class_object_manager}{Object\+Manager} and updated/rendered automatically.
