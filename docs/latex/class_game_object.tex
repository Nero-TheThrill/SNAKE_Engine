\doxysubsection{Detailed Description}
Base interface for all game world objects. 

\doxylink{class_game_object}{Game\+Object} is the root abstract class for all entities that exist in the game world, such as players, enemies, bullets, or UI elements.

Each object follows a standard lifecycle consisting of\+:
\begin{DoxyItemize}
\item \doxylink{class_game_object_ad115e2b3f240578d8ebd04e994141e63}{Init()}\+: Initialize this object
\item \doxylink{class_game_object_a69148215e3765d3c2e75ee09df7c0e6e}{Late\+Init()}\+: Cross-\/object setup after all objects are registered
\item \doxylink{class_game_object_a5e5d39ae8bf189946dc372a5b8895006}{Update()}\+: Per-\/frame game logic
\item \doxylink{class_game_object_a9d3fa8a1c64815a19c4b5540478584a0}{Draw()}\+: Per-\/frame rendering
\item \doxylink{class_game_object_afb67ad7dccc69500d26d318a12a00c69}{Free()}\+: Cleanup before removal
\item \doxylink{class_game_object_aeb5e2de10ad0afc74a5cedec5673b0e1}{Late\+Free()}\+: Final cleanup after object dependencies are unlinked
\end{DoxyItemize}

Game\+Objects are managed by \doxylink{class_object_manager}{Object\+Manager} and tagged via string identifiers.
