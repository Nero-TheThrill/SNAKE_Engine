\doxysubsection{Detailed Description}
Minimal \doxylink{class_object}{Object} subclass with no-\/op lifecycle methods. 


\begin{DoxyItemize}
\item Constructs the base \doxylink{class_object}{Object} with \doxylink{_object_8h_a842c5e2e69277690b064bf363c017980a4504e1ed59cd9732b8a844e5424e6f13}{Object\+Type\+::\+GAME}.
\item All lifecycle hooks (Init/\+Late\+Init/\+Update/\+Draw/\+Free/\+Late\+Free) are empty by default, so users can override only the ones they need. 
\end{DoxyItemize}
