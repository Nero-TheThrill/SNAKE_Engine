\doxysubsection{Detailed Description}
Represents a 2D texture loaded from file or raw image data. 

The \doxylink{class_texture}{Texture} class encapsulates an Open\+GL texture object, storing its dimensions, format, and Open\+GL-\/assigned ID. It supports configurable filtering, wrapping, and mipmap generation via the \doxylink{struct_texture_settings}{Texture\+Settings} struct.

Textures can be loaded either from image files or directly from memory. Once created, textures can be bound to specific texture units for use in shaders.

\begin{DoxyNote}{Note}
\doxylink{class_texture}{Texture} instances are managed internally by \doxylink{class_render_manager}{Render\+Manager} and used in \doxylink{class_material}{Material} bindings. 
\end{DoxyNote}
