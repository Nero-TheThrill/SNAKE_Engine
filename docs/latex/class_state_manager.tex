\doxysubsection{Detailed Description}
Manages the active \doxylink{class_game_state}{Game\+State} and orchestrates state transitions. 


\begin{DoxyItemize}
\item Holds the current state and a queued next state.
\item The actual transition occurs in \doxylink{class_state_manager_ab4ebd3189a4623e805d95f768b314acb}{Update()}\+: the previous state is freed/unloaded, then the next state is loaded/initialized and becomes current.
\item On transition, all sounds are stopped via Sound\+Manager\+::\+Control\+All(\+Stop).
\item After a state\textquotesingle{}s \doxylink{class_state_manager_ae161e57265af366c58b0e2b5d936553d}{Draw()}, \doxylink{class_render_manager_a620231f7f4119dbf26ef416478578cde}{Render\+Manager\+::\+Flush\+Draw\+Commands()} is called; if the engine requests debug draw, Flush\+Debug\+Line\+Draw\+Commands() is also called.
\item On transition, the new state\textquotesingle{}s \doxylink{class_camera_manager}{Camera\+Manager} receives the current window size via \doxylink{class_camera_manager_a2d94c81e097f06d18ffbfd7a38ef8f48}{Camera\+Manager\+::\+Set\+Screen\+Size\+For\+All}(\doxylink{class_window_manager_ad8e16ad88c8eb3e3889d5c3dbe7891cb}{Window\+Manager\+::\+Get\+Width}/\+Height()).
\end{DoxyItemize}

\begin{DoxyNote}{Note}
\doxylink{class_state_manager_ab4ebd3189a4623e805d95f768b314acb}{Update()}, \doxylink{class_state_manager_ae161e57265af366c58b0e2b5d936553d}{Draw()}, \doxylink{class_state_manager_aa8e5c183bf725b7132f7e4bc800deaab}{Free()} are internal (called by \doxylink{class_s_n_a_k_e___engine}{SNAKE\+\_\+\+Engine}). 
\end{DoxyNote}

Per-\/frame update and (if queued) state transition. 

If next\+State exists\+:
\begin{DoxyItemize}
\item If current\+State exists, calls current\+State-\/\texorpdfstring{$>$}{>}System\+Free() and System\+Unload(), then engine\+Context.\+sound\+Manager-\/\texorpdfstring{$>$}{>}Control\+All(\+Sound\+Manager\+::\+Sound\+Control\+Type\+::\+Stop).
\item Moves next\+State into current\+State.
\item Calls current\+State-\/\texorpdfstring{$>$}{>}System\+Load(engine\+Context) and System\+Init(engine\+Context).
\item Calls current\+State-\/\texorpdfstring{$>$}{>}Get\+Camera\+Manager().Set\+Screen\+Size\+For\+All( engine\+Context.\+window\+Manager-\/\texorpdfstring{$>$}{>}Get\+Width(), engine\+Context.\+window\+Manager-\/\texorpdfstring{$>$}{>}Get\+Height()). \end{DoxyItemize}
