\doxysubsection{Detailed Description}
Free\+Type-\/based bitmap font with dynamic atlas and text mesh generation. 


\begin{DoxyItemize}
\item Constructor loads a TTF via Free\+Type and creates a single-\/channel (R) atlas texture, then initializes a \doxylink{class_material}{Material} using shader tag "{}\mbox{[}\+Engine\+Shader\mbox{]}internal\+\_\+text"{}. The material binds the atlas as "{}u\+\_\+\+Font\+Texture"{} and sets default "{}u\+\_\+\+Color"{} = (1,1,1,1).
\item Atlas starts at 128x128 and expands by doubling; all baked glyphs are re-\/packed on expand.
\item Text input is parsed as UTF-\/8; per-\/glyph advances are used (no kerning).
\item Newlines (\textquotesingle{}~\newline
\textquotesingle{}) start a new line; overall size = (max line width, font\+Size x line\+Count). 
\end{DoxyItemize}
