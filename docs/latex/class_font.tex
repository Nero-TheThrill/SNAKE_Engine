\doxysubsection{Detailed Description}
A font rendering class that supports UTF-\/8 text and dynamic glyph atlas baking. 

The \doxylink{class_font}{Font} class encapsulates font loading, glyph caching, and mesh generation for text rendering. It uses Free\+Type to rasterize glyphs on demand and dynamically packs them into a single texture atlas.

Each \doxylink{class_font}{Font} instance maintains its own texture atlas and material. It also supports horizontal and vertical alignment for multi-\/line text layout. Text meshes are typically generated internally by \doxylink{class_text_object}{Text\+Object} or \doxylink{struct_text_instance}{Text\+Instance}, not by user code.

This class is not constructed directly by users. Use \doxylink{class_render_manager_a0762465f3bfa7573ec3261767a68eb16}{Render\+Manager\+::\+Register\+Font()} to register a font, and \doxylink{class_render_manager_a1db9a464555a13cecb1a865f92eae85c}{Render\+Manager\+::\+Get\+Font\+By\+Tag()} to retrieve it.

The font supports both ASCII and full Unicode input (including Hangul and CJK characters), and dynamically expands its atlas if necessary via \doxylink{class_font_a225429b93aa4fe0d76b2fecd9ac88142}{Expand\+Atlas()}.

\begin{DoxyNote}{Note}
In most cases, you do not need to call \doxylink{class_font_a3a3963ea1c039525cbf922f52ae03670}{Generate\+Text\+Mesh()} directly -\/ \doxylink{class_text_object}{Text\+Object} and \doxylink{struct_text_instance}{Text\+Instance} internally manage mesh generation and caching.
\end{DoxyNote}

Generates a mesh for rendering a given UTF-\/8 string with specified alignment. 

This function takes a UTF-\/8 encoded string and constructs a quad-\/based mesh that visually represents the text using the current font\textquotesingle{}s atlas. Each character is rendered as a pair of triangles (a quad), using the glyph\textquotesingle{}s UV coordinates and pixel size.

Multi-\/line support\+:
\begin{DoxyItemize}
\item Line breaks (\textquotesingle{}~\newline
\textquotesingle{}) are respected and processed.
\item Each line is measured individually, and text is aligned based on the specified \textquotesingle{}\doxylink{_font_8h_adba6445b3653352a2f932f18e80c4c44}{Text\+AlignH}\textquotesingle{} (horizontal) and \textquotesingle{}\doxylink{_font_8h_adddce2e8703c25cb1fda789220605572}{Text\+AlignV}\textquotesingle{} (vertical) modes.
\end{DoxyItemize}

\doxylink{struct_glyph}{Glyph} handling\+:
\begin{DoxyItemize}
\item For each character in the input, the glyph is first checked in the atlas.
\item If the glyph is missing, it is baked on-\/demand using \doxylink{class_font_aa3f349ce0e57bfabb0ef4cea1431a1d6}{Try\+Bake\+Glyph()}.
\item Glyphs with no bitmap or advance are skipped or replaced with a fallback (\textquotesingle{}?\textquotesingle{}) glyph.
\end{DoxyItemize}

The resulting mesh includes vertices with positions and UVs, and indices forming triangles for each character. It supports batching, alignment, and is compatible with the engine\textquotesingle{}s instanced or non-\/instanced rendering.
