\doxysubsection{Detailed Description}
Manages a set of named \doxylink{class_camera2_d}{Camera2D} instances and an active camera tag. 


\begin{DoxyItemize}
\item Cameras are owned via std\+::unique\+\_\+ptr and stored by string tag.
\item Registering with an existing tag overwrites the previous camera.
\item Active camera changes only if the tag exists.
\item \doxylink{class_camera_manager_ace05611218ec9a7e43db3e658b51f404}{Clear()} empties the registry but does not reset the active tag string. 
\end{DoxyItemize}
