\doxysubsection{Detailed Description}
Logging and debugging macro definitions for the SNAKE engine. 

This file defines macros for standardized logging output in debug builds, including\+:
\begin{DoxyItemize}
\item SNAKE\+\_\+\+LOG\+: general purpose log messages
\item SNAKE\+\_\+\+WRN\+: warning messages
\item SNAKE\+\_\+\+ERR\+: error messages
\end{DoxyItemize}

Logging macros include the filename and line number of the call site. You can selectively disable specific levels of output by defining\+:
\begin{DoxyItemize}
\item SKIP\+\_\+\+LOG\+: disables SNAKE\+\_\+\+LOG
\item SKIP\+\_\+\+WRN\+: disables SNAKE\+\_\+\+WRN
\item SKIP\+\_\+\+ERR\+: disables SNAKE\+\_\+\+ERR
\end{DoxyItemize}

These macros are no-\/ops in builds unless SNAKE\+\_\+\+DEBUG is defined.
