\doxysubsection{Detailed Description}
Simple logging macros for debug output. 

Defines macros for logging messages to the console. These macros automatically include the filename and line number, and are only enabled in debug builds ({\ttfamily \+\_\+\+DEBUG} is defined).

In release mode, all macros are compiled out with no overhead.
