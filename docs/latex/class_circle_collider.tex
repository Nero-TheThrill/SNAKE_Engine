\doxysubsection{Detailed Description}
Circle collider with radius (size = 2xradius). 


\begin{DoxyItemize}
\item Maintains base (unscaled) radius and a scaled radius cache.
\item When transform-\/scale coupling is enabled, \doxylink{class_circle_collider_a8f00aeea30fb3ffb45ebbcaab0d96fdd}{Sync\+With\+Transform\+Scale()} updates the effective radius from the owner\textquotesingle{}s scale.
