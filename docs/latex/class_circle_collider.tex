\doxysubsection{Detailed Description}
A circular 2D collider component. 

\doxylink{class_circle_collider}{Circle\+Collider} is a concrete subclass of \doxylink{class_collider}{Collider} representing a circle shape. It stores both base radius and scaled radius, and supports scaling via transform.

It provides collision resolution against both Circle and AABB shapes. The center of the collider is world-\/aligned based on the owning object\textquotesingle{}s transform.

\begin{DoxyNote}{Note}
Used internally by \doxylink{class_object_manager}{Object\+Manager}. Do not instantiate directly. 
\end{DoxyNote}
