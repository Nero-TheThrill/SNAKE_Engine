\doxysubsection{Detailed Description}
Lightweight bundle of non-\/owning pointers to core engine systems. 


\begin{DoxyItemize}
\item Populated in \doxylink{class_s_n_a_k_e___engine_aa6891c4512199870f8ee25cc6d82e45a}{SNAKE\+\_\+\+Engine\+::\+Set\+Engine\+Context()}.

\begin{DoxyItemize}
\item In Draw()\+: calls Render\+Manager\+::\+Flush\+Draw\+Commands(engine\+Context) and, if engine.\+Should\+Render\+Debug\+Draws(), calls Render\+Manager\+::\+Flush\+Debug\+Line\+Draw\+Commands(engine\+Context).
\item In Free()\+: forwards to the current state\textquotesingle{}s System\+Free/\+System\+Unload.
\end{DoxyItemize}
\item Used by the GLFW framebuffer-\/size callback (\doxylink{_window_manager_8cpp}{Window\+Manager.\+cpp})\+:

\item Updates \doxylink{class_window_manager_ac822306fe0eb51630f31874ec6c469a4}{Window\+Manager\+::\+Set\+Width}/\+Set\+Height().
\item Propagates size to the current state\textquotesingle{}s Camera\+Manager\+::\+Set\+Screen\+Size\+For\+All(width, height).
\end{DoxyItemize}
\end{DoxyItemize}
