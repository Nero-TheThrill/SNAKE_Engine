\doxysubsection{Detailed Description}
Core context struct that provides access to all major engine systems. 

\doxylink{struct_engine_context}{Engine\+Context} is a central struct used throughout the engine to pass references to major subsystems such as state, rendering, input, sound, and window management. This context is typically set once during engine initialization in \doxylink{class_s_n_a_k_e___engine_aa6891c4512199870f8ee25cc6d82e45a}{SNAKE\+\_\+\+Engine\+::\+Set\+Engine\+Context()} and passed to systems such as \doxylink{class_object_manager}{Object\+Manager}, \doxylink{class_game_state}{Game\+State}, and others.

This pattern helps avoid global variables and simplifies dependency injection.

\begin{DoxyNote}{Note}
Should be initialized once after window creation and reused across all systems. 
\end{DoxyNote}
