\doxysubsection{Detailed Description}
Shared access point to all major engine systems. 

Acts as a container that holds pointers to engine-\/wide subsystems\+:
\begin{DoxyItemize}
\item \doxylink{class_state_manager}{State\+Manager}
\item \doxylink{class_window_manager}{Window\+Manager}
\item \doxylink{class_input_manager}{Input\+Manager}
\item \doxylink{class_render_manager}{Render\+Manager}
\item Root engine instance (\doxylink{class_s_n_a_k_e___engine}{SNAKE\+\_\+\+Engine})
\end{DoxyItemize}

This struct is owned and initialized by the engine, and passed to game systems like \doxylink{class_game_state}{Game\+State}, \doxylink{class_game_object}{Game\+Object}, and custom logic that need coordinated access to rendering, input, or state transitions.
