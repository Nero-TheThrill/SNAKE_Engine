\doxysubsection{Detailed Description}
A 2D collider with axis-\/aligned bounding box shape. 

\doxylink{class_a_a_b_b_collider}{AABBCollider} stores half-\/size in both base and scaled forms and performs collision detection using axis-\/aligned rectangular bounds.

Supports transform-\/based scaling and collision with both AABB and Circle colliders. The center is aligned with the owner\textquotesingle{}s world position.
