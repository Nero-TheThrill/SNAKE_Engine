\doxysubsection{Detailed Description}
Axis-\/aligned bounding-\/box collider, stored as half-\/size. 


\begin{DoxyItemize}
\item Keeps base (unscaled) half-\/size and a scaled half-\/size cache.
\item When transform-\/scale coupling is enabled, \doxylink{class_a_a_b_b_collider_a66dc8ecf507a6f3de51fb56f7cd0aa67}{Sync\+With\+Transform\+Scale()} updates the effective half-\/size from the owner\textquotesingle{}s scale. 
\end{DoxyItemize}
