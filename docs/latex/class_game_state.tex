\doxysubsection{Detailed Description}
Base class for a game state (scene/screen). 


\begin{DoxyItemize}
\item Owns an \doxylink{class_object_manager}{Object\+Manager} and a \doxylink{class_camera_manager}{Camera\+Manager}.
\item Override the protected virtual hooks (Init/\+Late\+Init/\+Update/\+Late\+Update/\+Load/\+Draw/\+Free/\+Unload) to implement state behavior.
\item Internal "{}\+System\texorpdfstring{$\ast$}{*}"{} methods coordinate the call order and are used by \doxylink{class_state_manager}{State\+Manager}. 
\end{DoxyItemize}
