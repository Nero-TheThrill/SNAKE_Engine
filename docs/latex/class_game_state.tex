\doxysubsection{Detailed Description}
Abstract base class representing a game state or scene. 

A \doxylink{class_game_state}{Game\+State} represents a self-\/contained mode in the game such as title screen, gameplay, or pause. It defines a full lifecycle\+: Load -\/\texorpdfstring{$>$}{>} Init -\/\texorpdfstring{$>$}{>} Update/\+Draw -\/\texorpdfstring{$>$}{>} Free -\/\texorpdfstring{$>$}{>} Unload. Each state manages its own \doxylink{class_object_manager}{Object\+Manager} and \doxylink{class_camera_manager}{Camera\+Manager}.

Override protected methods to implement the specific logic of your game state.
