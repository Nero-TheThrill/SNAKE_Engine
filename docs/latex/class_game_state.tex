\doxysubsection{Detailed Description}
Base class for representing a single game state or scene. 

\doxylink{class_game_state}{Game\+State} defines the standard lifecycle for a self-\/contained game state such as a title screen, gameplay level, or pause menu. It encapsulates its own \doxylink{class_object_manager}{Object\+Manager} and \doxylink{class_camera_manager}{Camera\+Manager}, and provides virtual methods for initialization, updates, rendering, loading, and cleanup.

The engine\textquotesingle{}s \doxylink{class_state_manager}{State\+Manager} controls which \doxylink{class_game_state}{Game\+State} is active and invokes its lifecycle methods\+:
\begin{DoxyItemize}
\item \doxylink{class_game_state_a89713af22cf5b07b602c3315b1f90296}{Load()} -\/\texorpdfstring{$>$}{>} \doxylink{class_game_state_ab4bdccf42209d9d3465a33739ad59646}{Init()} -\/\texorpdfstring{$>$}{>} \doxylink{class_game_state_a6d11e92e629e2d4cbba792faeb75a71f}{Update()} / \doxylink{class_game_state_ab3f79ba850091b4853d800b1ff2e2974}{Draw()} -\/\texorpdfstring{$>$}{>} \doxylink{class_game_state_a46b0b108e44fa42845f27d20bc242d93}{Free()} -\/\texorpdfstring{$>$}{>} \doxylink{class_game_state_ab9a398de72a4201a69093dfd574220c6}{Unload()}
\end{DoxyItemize}

Users should inherit from \doxylink{class_game_state}{Game\+State} and override these methods to define scene-\/specific behavior. \doxylink{class_object_manager}{Object\+Manager} and \doxylink{class_camera_manager}{Camera\+Manager} are scoped per state, allowing clean separation of logic.

\begin{DoxyNote}{Note}
System-\/level lifecycle methods like \doxylink{class_game_state_a471d08b2632c8696a1105abf356d11e1}{System\+Init()} and \doxylink{class_game_state_a6264f6b70aa45b3e72b6067061cc2395}{System\+Update()} are managed by \doxylink{class_state_manager}{State\+Manager} and should not be overridden directly.
\end{DoxyNote}
