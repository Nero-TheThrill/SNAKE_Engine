\doxysubsection{Detailed Description}
\doxylink{class_material}{Material} binds a \doxylink{class_shader}{Shader} with named textures and cached uniform values. 


\begin{DoxyItemize}
\item Textures are associated by uniform name (e.\+g., "{}u\+\_\+\+Texture"{}).
\item Uniform values are staged via \doxylink{class_material_a6c49cb3082a29e0d412577dad3cfde5f}{Set\+Uniform()} and sent to the GPU on \doxylink{class_material_a39001e95de2408283f42efa7e8657a05}{Send\+Uniforms()}.
\item \doxylink{class_render_manager}{Render\+Manager} calls \doxylink{class_material_a1fd6c6c16d1f04e747470a2cc0c52fa1}{Bind()}/\+Send\+Uniforms()/\+Un\+Bind() during submission; users typically only call \doxylink{class_material_af587d5938f543479d1d2955cb4da1f48}{Set\+Texture()}/\+Set\+Uniform().
\item Instancing\+: \doxylink{class_material_a7a9c710b2d7a286c1349315b6fd6ab7a}{Is\+Instancing\+Supported()} is true only if instancing is enabled on this material and the underlying shader reports support. 
\end{DoxyItemize}
