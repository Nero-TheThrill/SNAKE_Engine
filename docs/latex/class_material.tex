\doxysubsection{Detailed Description}
Represents a GPU-\/bound material used to render objects. 

A \doxylink{class_material}{Material} encapsulates a \doxylink{class_shader}{Shader}, a set of bound textures, and uniform values. It also optionally supports GPU instancing using a per-\/instance matrix buffer.


\begin{DoxyItemize}
\item Textures are set using \doxylink{class_material_af587d5938f543479d1d2955cb4da1f48}{Set\+Texture()}.
\item Uniforms (int, float, vectors, matrices) are set using \doxylink{class_material_a6c49cb3082a29e0d412577dad3cfde5f}{Set\+Uniform()}.
\item Call \doxylink{class_material_ae790a5b6e8836e871f6e01f4431350dd}{Enable\+Instancing()} to enable per-\/instance rendering.
\end{DoxyItemize}

Used internally by \doxylink{class_object_manager}{Object\+Manager} and \doxylink{class_render_manager}{Render\+Manager} to prepare draw submissions.
