\doxysubsection{Detailed Description}
Represents a GPU-\/bound material that links a shader, textures, and uniforms. 

The \doxylink{class_material}{Material} class encapsulates how an object should appear when rendered. It manages a reference to a \doxylink{class_shader}{Shader}, associated textures (by uniform name), and arbitrary uniform values (int, float, vecX, mat4, etc.).

Materials can optionally enable hardware instancing, in which case instance data such as transforms and per-\/instance uniforms are sent via VBOs.

\begin{DoxyNote}{Note}
Materials are usually created and managed by the \doxylink{class_render_manager}{Render\+Manager}, and assigned to renderable objects such as \doxylink{class_mesh}{Mesh} or \doxylink{class_object}{Object}. 
\end{DoxyNote}
