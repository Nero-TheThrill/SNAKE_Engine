Returns the 4x4 transform matrix (lazy-\/recomputed if dirty). 

Rebuilds the matrix when \textquotesingle{}is\+Changed\textquotesingle{} is true using\+: T = translate(position.\+x, position.\+y, 0) R = rotate around Z by \textquotesingle{}rotation\textquotesingle{} (as required by glm\+::rotate) S = scale(scale.\+x, scale.\+y, 1) Final composition\+: matrix = T \texorpdfstring{$\ast$}{*} R \texorpdfstring{$\ast$}{*} S; then clears \textquotesingle{}is\+Changed\textquotesingle{}.

\begin{DoxyReturn}{Returns}
