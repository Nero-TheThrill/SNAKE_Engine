\doxysubsection{Detailed Description}
Represents a 2D transformation including position, rotation, and scale. 

\doxylink{class_transform2_d}{Transform2D} stores 2D spatial information used by game objects such as position, rotation, and scale. It computes a model matrix on demand when changes occur, and avoids recomputation unless necessary. This class is widely used in rendering, physics, and animation systems to represent object transforms.

\begin{DoxyNote}{Note}
All matrix updates are lazy; \doxylink{class_transform2_d_a86e9a6ffac2c382a9e3467f8331bf86d}{Get\+Matrix()} triggers recalculation only if changes were made. 
\end{DoxyNote}
