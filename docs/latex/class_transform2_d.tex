\doxysubsection{Detailed Description}
Represents a 2D spatial transform with position, rotation, and scale. 

Used to define an object\textquotesingle{}s position in world space, orientation, and scale. Generates a 4x4 matrix for rendering, recomputed lazily when any value changes.

Example usage\+: 
\begin{DoxyCode}{0}
\DoxyCodeLine{transform.SetPosition(\{100.f,\ 200.f\});}
\DoxyCodeLine{glm::mat4\ model\ =\ transform.GetMatrix();}

\end{DoxyCode}
