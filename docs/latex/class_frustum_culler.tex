\doxysubsection{Detailed Description}
Utility class for view frustum culling based on 2D bounding circles. 

\doxylink{class_frustum_culler}{Frustum\+Culler} performs visibility checks for a list of Game\+Objects using simple 2D bounding radius tests against the camera\textquotesingle{}s visible area. This helps improve performance by skipping rendering of off-\/screen objects.
