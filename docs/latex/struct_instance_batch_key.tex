\doxysubsection{Detailed Description}
Represents a unique pair of mesh and material for batching. 

This struct is used as a key when grouping Game\+Objects that share both a \doxylink{class_mesh}{Mesh} and a \doxylink{class_material}{Material}. Such groups can be rendered together efficiently using instancing. Comparison and ordering operators are provided for use in maps and sets.
