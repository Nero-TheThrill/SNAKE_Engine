\doxysubsection{Detailed Description}
Key used to batch draw calls for instanced rendering. 

Two objects can be batched together when they share the same \doxylink{class_mesh}{Mesh}, \doxylink{class_material}{Material}, and \doxylink{class_sprite_sheet}{Sprite\+Sheet} pointers. This key groups such objects.
