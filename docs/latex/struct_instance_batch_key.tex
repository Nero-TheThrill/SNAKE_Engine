\doxysubsection{Detailed Description}
Represents a unique key for grouping objects by mesh and material. 

This struct is used to group Game\+Objects that share the same \doxylink{class_mesh}{Mesh} and \doxylink{class_material}{Material}. It serves as a key in containers such as std\+::map or std\+::unordered\+\_\+map, allowing instanced rendering to efficiently batch draw calls with minimal GPU state changes.

The comparison operators (\textquotesingle{}==\textquotesingle{} and \textquotesingle{}\texorpdfstring{$<$}{<}\textquotesingle{}) allow it to be used in both hashed and ordered containers. Note that pointer identity is used for comparison-\/objects must share exact \doxylink{class_mesh}{Mesh} and \doxylink{class_material}{Material} instances.

\begin{DoxyNote}{Note}
Used internally in \doxylink{class_render_manager}{Render\+Manager}\textquotesingle{}s instanced rendering pipeline. 
\end{DoxyNote}
