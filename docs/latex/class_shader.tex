\doxysubsection{Detailed Description}
Represents an Open\+GL shader program composed of one or more stages. 

The \doxylink{class_shader}{Shader} class manages the lifecycle of an Open\+GL program, including attaching shader stages (e.\+g., vertex, fragment), compiling source code, and linking them into a usable GPU program.

It provides uniform upload functions and tracks whether the shader supports hardware instancing based on its input layout.

Shaders are typically registered through \doxylink{class_render_manager}{Render\+Manager} and bound automatically during material usage.

\begin{DoxyNote}{Note}
\doxylink{class_shader}{Shader} instances are owned and used by \doxylink{class_render_manager}{Render\+Manager} and \doxylink{class_material}{Material}. Users typically do not create or bind shaders directly in game code. 
\end{DoxyNote}
