\doxysubsection{Detailed Description}
Open\+GL shader program wrapper with uniform helpers and instancing support query. 


\begin{DoxyItemize}
\item Constructor creates a program (gl\+Create\+Program). Destructor deletes attached shaders and the program.
\item Attach\+From\+File/\+Attach\+From\+Source compile a stage and attach it to the program.
\item \doxylink{class_shader_a4d19cc34a5294ae473d624bd86630cbf}{Link()} links the program; enforces TCS/\+TES pairing; then checks instancing support via attribute "{}i\+\_\+\+Model"{}.
\item Send\+Uniform\texorpdfstring{$\ast$}{*}() resolves a uniform location each call; logs if not found; then uploads the value.
\item \doxylink{class_material}{Material} calls \doxylink{class_shader_a11a2de17d1d077ec34acd4f7cd57f6b1}{Use()}/\+Unuse() and Send\+Uniform\texorpdfstring{$\ast$}{*}() through friendship; \doxylink{class_render_manager}{Render\+Manager} may also access as a friend. 
\end{DoxyItemize}
