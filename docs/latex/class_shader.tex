\doxysubsection{Detailed Description}
Represents an Open\+GL shader program composed of multiple shader stages. 

The \doxylink{class_shader}{Shader} class handles loading, compiling, linking, and using GPU shader programs. It supports uniform uploading and optional instancing checks. Shaders can be built from external source files, one per stage.

Typical usage includes loading a shader, linking, and then binding it before rendering.
