\doxysubsection{Detailed Description}
Central renderer and asset registry for GPU resources. 

\doxylink{class_render_manager}{Render\+Manager} handles registration and lookup for all GPU-\/bound assets including shaders, meshes, textures, and materials. It also manages per-\/frame draw submission, and issuing render commands to the GPU.

The rendering flow consists of\+:
\begin{DoxyItemize}
\item \doxylink{class_render_manager_a5906c7d7d3fbcece5fe4851c520d3154}{Submit()}\+: collects draw calls
\item \doxylink{class_render_manager_aff85070eef9769739f2478fd5b86da87}{Flush\+Draw\+Commands()}\+: executes all sorted draw calls 
\end{DoxyItemize}
