Flushes all queued draw batches to the GPU. 


\begin{DoxyItemize}
\item Iterates layers -\/\texorpdfstring{$>$}{>} shaders -\/\texorpdfstring{$>$}{>} (mesh, material, spritesheet) batches.
\item If the first object in a batch is instancing-\/capable, submits per-\/instance data; otherwise draws objects one-\/by-\/one.
\item Sets view/projection from the batch camera (or identity view); falls back to engine window size when no camera is present.
\item Ensures a bound texture exists; if not, binds the engine error texture.
\item Clears the internal render map afterwards.
