\doxysubsection{Detailed Description}
Simple frame animator operating on a \doxylink{class_sprite_sheet}{Sprite\+Sheet}. 


\begin{DoxyItemize}
\item Supports playing a direct frame range or a named clip from the sheet.
\item Update(dt) advances time and steps the current frame according to the active range\textquotesingle{}s frame\+Time or the clip\textquotesingle{}s frame\+Duration/looping. 
\end{DoxyItemize}
