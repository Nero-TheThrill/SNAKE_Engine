\doxysubsection{Detailed Description}
Controls playback of sprite animations using a \doxylink{class_sprite_sheet}{Sprite\+Sheet}. 

\doxylink{class_sprite_animator}{Sprite\+Animator} handles frame-\/by-\/frame animation using either\+:
\begin{DoxyItemize}
\item A direct frame range (via Play\+Clip(start, end))
\item A named clip from the \doxylink{class_sprite_sheet}{Sprite\+Sheet} (via Play\+Clip("{}clip\+Name"{}))
\end{DoxyItemize}

On each Update(dt), it advances the animation timer and determines the current frame. The UV offset and scale are updated per frame and can be queried for rendering.

This class does not directly render anything; it simply manages which part of the sprite sheet should be shown and for how long.

\begin{DoxyNote}{Note}
Usually attached to animated objects (e.\+g. \doxylink{class_game_object}{Game\+Object} or \doxylink{class_text_object}{Text\+Object}). 
\end{DoxyNote}
