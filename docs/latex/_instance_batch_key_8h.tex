Type alias for organizing Game\+Objects into instancing-\/compatible groups. 

This two-\/level map groups Game\+Objects first by render layer (int), then by unique mesh-\/material pair (\doxylink{struct_instance_batch_key}{Instance\+Batch\+Key}). Used by \doxylink{class_render_manager}{Render\+Manager} to batch render instances efficiently.

Structure\+:
\begin{DoxyItemize}
\item outer key\+: render layer (e.\+g., UI, Background)
\item inner key\+: \doxylink{struct_instance_batch_key}{Instance\+Batch\+Key} (mesh + material)
\item value\+: list of \doxylink{class_game_object}{Game\+Object} pointers to render
\end{DoxyItemize}

\begin{DoxyNote}{Note}
This structure is rebuilt every frame before instanced draw calls. 
\end{DoxyNote}
