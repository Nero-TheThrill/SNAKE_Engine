\doxysubsection{Detailed Description}
A 2D camera class used for orthographic rendering and view culling. 

The \doxylink{class_camera2_d}{Camera2D} class provides basic functionality for controlling the visible area of a 2D scene. It supports panning, zooming, and determining if an object is within the camera\textquotesingle{}s current viewport. The projection matrix is currently constructed using orthographic projection based on the screen size and zoom level.

This camera is primarily used during rendering for calculating the visible region and performing frustum culling via \doxylink{class_camera2_d_ac59db8b285e9e7c9646b6e369cfda825}{Is\+In\+View()}.

\begin{DoxyNote}{Note}
The current implementation returns an orthographic projection matrix via \doxylink{class_camera2_d_a01c3bcd77bfc25fa22b519edfb56a45b}{Get\+Projection\+Matrix()}. In a future update, this function will be removed and replaced with Get\+View\+Matrix().
\end{DoxyNote}
