\doxysubsection{Detailed Description}
A simple 2D orthographic camera for screen-\/space rendering. 

\doxylink{class_camera2_d}{Camera2D} handles position, zoom, and projection matrix generation. It also provides 2D frustum culling via bounding-\/circle intersection.
