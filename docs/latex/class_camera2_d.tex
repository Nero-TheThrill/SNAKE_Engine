\doxysubsection{Detailed Description}
Simple 2D camera storing position, zoom, and screen size. 

Provides a view matrix and a basic visibility test against a circular bound. Screen width/height are stored for reference; callers pass a viewport size to \doxylink{class_camera2_d_ac59db8b285e9e7c9646b6e369cfda825}{Is\+In\+View()} when performing culling.
