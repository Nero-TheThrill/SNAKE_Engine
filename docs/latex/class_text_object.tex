\doxysubsection{Detailed Description}
A 2D object that renders UTF-\/8 text on screen. 

\doxylink{class_text_object}{Text\+Object} is a renderable entity designed to display dynamic or static text using a specified \doxylink{class_font}{Font}. It generates and caches mesh geometry based on the text content and alignment options, supporting per-\/frame updates and world transformation.

This class does not support animation or material/mesh customization. It uses \doxylink{struct_text_instance}{Text\+Instance} internally to manage font and text content, and supports horizontal and vertical alignment.

\begin{DoxyNote}{Note}
\doxylink{class_text_object}{Text\+Object} cannot be assigned a custom material or mesh. It is automatically updated and batched by the rendering system. 
\end{DoxyNote}
