\doxysubsection{Detailed Description}
\doxylink{class_object}{Object} subclass that renders text using a Font-\/generated mesh. 


\begin{DoxyItemize}
\item Constructor sets material = font-\/\texorpdfstring{$>$}{>}\doxylink{class_text_object_a2ddeaaee422f88c2ac6251119dfc25fd}{Get\+Material()} and builds an initial mesh via \doxylink{class_text_object_a0705bf6f2f40788da1c9fb60368c183b}{Update\+Mesh()}.
\item \doxylink{class_mesh}{Mesh} is (re)built from Font\+::\+Generate\+Text\+Mesh(text, align\+H, align\+V).
\item \doxylink{class_text_object_a534ee1073bdc4b0b2d953dfbe8166f03}{Draw()} batching/flush is handled by \doxylink{class_render_manager}{Render\+Manager}.
\item Alignment affects world-\/space anchoring; Get\+World\+Position/\+Get\+World\+Scale override \doxylink{class_object}{Object}\textquotesingle{}s behavior.
\item Material/\+Mesh setters and animator APIs are deleted for this class. 
\end{DoxyItemize}
