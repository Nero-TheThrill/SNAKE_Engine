\doxysubsection{Detailed Description}
Stores texture sampling behavior including filter and wrap modes. 

Defines how a texture should be filtered (minification/magnification) and wrapped (for out-\/of-\/bounds coordinates). Passed as a configuration object when loading a \doxylink{class_texture}{Texture}.
