\doxysubsection{Detailed Description}
Loads, plays, and controls sounds; tracks channels and reusable instance IDs. 


\begin{DoxyItemize}
\item Public API\+:
\begin{DoxyItemize}
\item Load\+Sound(tag, filepath, loop)\+: loads a sound and stores it under {\ttfamily tag}.
\item Play(tag, volume, start\+Time\+Sec)\+: starts playback; returns a reusable \doxylink{_sound_manager_8h_a7675718ce8055fa8280410c0d32ad4f3}{Sound\+Instance\+ID}.
\item Set\+Volume\+By\+ID / Set\+Volume\+By\+Tag / Set\+Volume\+All\+: volume control.
\item Control\+By\+ID / Control\+By\+Tag / Control\+All\+: Pause, Resume, Stop.
\end{DoxyItemize}
\item Engine integration\+:
\begin{DoxyItemize}
\item \doxylink{class_s_n_a_k_e___engine}{SNAKE\+\_\+\+Engine} calls \doxylink{class_sound_manager_a79cda227ead66f1cc3cd429c8b998b08}{Init()} during startup, \doxylink{class_sound_manager_aaf241621221cdbefeba78e8b6bc29240}{Update()} each frame, and \doxylink{class_sound_manager_af25662c1d0237540686087a080001f9b}{Free()} on shutdown.
\end{DoxyItemize}
\item Internals\+:
\begin{DoxyItemize}
\item active\+Channels\mbox{[}tag\mbox{]} keeps FMOD\+::\+Channel\texorpdfstring{$\ast$}{*} for currently playing sounds by tag.
\item instance\+Map maps \doxylink{_sound_manager_8h_a7675718ce8055fa8280410c0d32ad4f3}{Sound\+Instance\+ID} -\/\texorpdfstring{$>$}{>} FMOD\+::\+Channel\texorpdfstring{$\ast$}{*}.
\item \doxylink{class_sound_manager_aaf241621221cdbefeba78e8b6bc29240}{Update()} calls \doxylink{class_sound_manager_a4a77d91e2c8257d6c2d02ffffad671ff}{Cleanup()} to remove finished channels and recycle IDs via reusable\+IDs. 
\end{DoxyItemize}
\end{DoxyItemize}
