\doxysubsection{Detailed Description}
Centralized audio playback and control system using FMOD. 

The \doxylink{class_sound_manager}{Sound\+Manager} is responsible for loading, playing, and controlling sounds within the engine. It wraps FMOD functionality and provides high-\/level interfaces for volume control, playback state management, and per-\/tag or per-\/instance sound operations.

Sounds are loaded by tag and played with a unique \doxylink{_sound_manager_8h_a7675718ce8055fa8280410c0d32ad4f3}{Sound\+Instance\+ID}, which can be reused. Volume and playback state (pause, resume, stop) can be controlled globally, by tag, or by instance.

\begin{DoxyNote}{Note}
This class is automatically initialized and updated by \doxylink{class_s_n_a_k_e___engine}{SNAKE\+\_\+\+Engine}. Users typically call \doxylink{class_sound_manager_acc7074efb4e8b5c71e85f556769d9b17}{Play()}, \doxylink{class_sound_manager_a8581621afa759742160371f27c31e6fa}{Set\+Volume\+By\+Tag()}, or \doxylink{class_sound_manager_aaa6392113f543f2bb32e6811bd010100}{Control\+By\+Tag()} from game logic. 
\end{DoxyNote}
