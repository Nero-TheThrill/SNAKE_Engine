\doxysubsection{Detailed Description}
Holds font and text content for rendering. 

\doxylink{struct_text_instance}{Text\+Instance} stores the font pointer and the actual UTF-\/8 text string to be displayed. It is used by \doxylink{class_text_object}{Text\+Object} to track what text to render and how it should be processed for mesh generation and alignment.

The \doxylink{struct_text_instance_ab47b24247b1607ba43bf9d84da6cf400}{Get\+Cache\+Key()} function is used internally to generate a key for caching text meshes.
