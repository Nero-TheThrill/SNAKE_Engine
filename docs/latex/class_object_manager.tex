\doxysubsection{Detailed Description}
Owns and orchestrates a collection of \doxylink{class_object}{Object} instances. 


\begin{DoxyItemize}
\item Accepts ownership of new objects, keeps a tag-\/\texorpdfstring{$>$}{>}object map, and exposes utility queries/draw helpers.
\item Provides lifecycle passes (Init/\+Update/\+Draw/\+Free) over managed objects.
\item Offers simple collision checking and collider debug-\/drawing. 
\end{DoxyItemize}
