\doxysubsection{Detailed Description}
Central manager for handling all \doxylink{class_object}{Object} instances in the engine. 

The \doxylink{class_object_manager}{Object\+Manager} is responsible for the creation, lifecycle management, rendering submission, and collision detection of Object-\/derived instances such as \doxylink{class_game_object}{Game\+Object} and \doxylink{class_text_object}{Text\+Object}.

It maintains internal containers to manage\+:
\begin{DoxyItemize}
\item {\ttfamily objects}\+: all active objects
\item {\ttfamily pending\+Objects}\+: newly added objects that will be initialized in the next update cycle
\item {\ttfamily raw\+Ptr\+Objects}\+: non-\/owning raw pointers for fast iteration
\item {\ttfamily object\+Map}\+: tag-\/based lookup for individual objects
\end{DoxyItemize}

Lifecycle is controlled via\+:
\begin{DoxyItemize}
\item {\ttfamily \doxylink{class_object_manager_ad7d870215c2f3033b6f2641e4c9dc8da}{Init\+All()}} and {\ttfamily Late\+Init()} for startup
\item {\ttfamily \doxylink{class_object_manager_a00c892376f4874c2f033a69526e18111}{Update\+All()}} for per-\/frame logic and animation update
\item {\ttfamily \doxylink{class_object_manager_a6150e34acf9947f2d2e36040d472df5f}{Erase\+Dead\+Objects()}} to remove killed objects (i.\+e., after {\ttfamily Kill()})
\item {\ttfamily \doxylink{class_object_manager_a2dcf63228b4ed4b45ae19304946f923f}{Free\+All()}} for full cleanup
\end{DoxyItemize}

Objects added via {\ttfamily \doxylink{class_object_manager_ae1f29824763903213d6044d463742db4}{Add\+Object()}} are automatically inserted into pending lists, and are initialized in the next {\ttfamily \doxylink{class_object_manager_a00c892376f4874c2f033a69526e18111}{Update\+All()}} call.

Collision is resolved using broad-\/phase grid and masking system. Each object is inserted into the spatial hash grid and checked using double dispatch.

\begin{DoxyNote}{Note}
This manager assumes ownership of all \doxylink{class_object}{Object} instances.
\end{DoxyNote}

Performs broad-\/phase and narrow-\/phase collision detection for all collidable objects. 

Uses a spatial hash grid to perform efficient broad-\/phase culling, followed by narrow-\/phase collision checks using each object\textquotesingle{}s \doxylink{class_collider}{Collider}. Collision responses are invoked via On\+Collision().

Objects must have valid \doxylink{class_collider}{Collider} instances and configured collision\+Category/mask via Set\+Collision().
