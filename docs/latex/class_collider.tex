\doxysubsection{Detailed Description}
Abstract collider attached to an \doxylink{class_object}{Object}, with optional transform-\/scale coupling. 


\begin{DoxyItemize}
\item Stores non-\/owning pointer to the owning \doxylink{class_object}{Object}.
\item World position can be set/read independently from the owner\textquotesingle{}s transform via \doxylink{class_collider_a41b2de089e3faf27bd0ac87d923f9203}{Set\+World\+Position()}/\+Get\+World\+Position().
\item If use\+Transform\+Scale is true, derived colliders may adapt their size to the owner\textquotesingle{}s scale (see \doxylink{class_collider_a188380ff8ece20474e86a662f995887e}{Sync\+With\+Transform\+Scale()}).
\item Collision uses double dispatch via \doxylink{class_collider_a6b1071065ccb7ddfc1ff8340a89246d7}{Dispatch\+Against()} overrides. 
\end{DoxyItemize}
