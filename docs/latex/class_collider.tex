\doxysubsection{Detailed Description}
Base class for all 2D collider shapes (circle, AABB) used in collision detection. 

\doxylink{class_collider}{Collider} is an abstract base class representing a 2D collision component. Each collider is owned by an \doxylink{class_object}{Object} and stores its world position independently from the transform.

Subclasses (\doxylink{class_circle_collider}{Circle\+Collider}, \doxylink{class_a_a_b_b_collider}{AABBCollider}) implement specific shape logic and double dispatch collision resolution. This class supports spatial hashing, debug drawing, and transform scaling.

\doxylink{class_collider}{Collider} instances are managed internally by \doxylink{class_object_manager}{Object\+Manager} and should not be created directly by users.

\begin{DoxyNote}{Note}
Collision checks use a double dispatch system via Check\+Collision and \doxylink{class_collider_a6b1071065ccb7ddfc1ff8340a89246d7}{Dispatch\+Against()}. 
\end{DoxyNote}
