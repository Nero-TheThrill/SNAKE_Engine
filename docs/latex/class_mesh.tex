\doxysubsection{Detailed Description}
Represents a GPU mesh composed of vertices and optional indices. 

The \doxylink{class_mesh}{Mesh} class manages vertex and index buffers (VBO, EBO) and their associated VAO. It supports multiple primitive types (e.\+g., triangles, lines, points), and enables both regular and instanced rendering through Open\+GL.

Meshes are constructed from a list of vertices and optional indices. The class also computes a local bounding box for efficient culling or layout.

\begin{DoxyNote}{Note}
\doxylink{class_mesh}{Mesh} instances are usually managed by \doxylink{class_render_manager}{Render\+Manager} and used by \doxylink{class_material}{Material}. 
\end{DoxyNote}
