\doxysubsection{Detailed Description}
Represents a GPU-\/side 2D mesh including vertex and optional index data. 

\doxylink{class_mesh}{Mesh} manages Open\+GL VAO/\+VBO/\+EBO buffers and provides draw functions for both normal and instanced rendering. It also computes and stores the local 2D bounding size of the mesh to support visibility culling.

\doxylink{class_mesh}{Mesh} instances are owned and drawn by \doxylink{class_render_manager}{Render\+Manager}.
