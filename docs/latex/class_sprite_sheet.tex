\doxysubsection{Detailed Description}
Represents a texture atlas subdivided into animation frames. 

\doxylink{class_sprite_sheet}{Sprite\+Sheet} stores frame layout information for a texture that is subdivided into equal-\/sized animation frames (grid-\/based). It computes per-\/frame UV coordinates and supports the definition of named animation clips.

This class is responsible for providing UV offset and scale for each frame, which are consumed by \doxylink{class_sprite_animator}{Sprite\+Animator} during playback. Clips can be defined using \doxylink{class_sprite_sheet_a0656d33c5b84e8b82abb34a7107ca823}{Add\+Clip()} and retrieved by name.

\begin{DoxyNote}{Note}
Used primarily for 2D sprite animation via instancing or mesh-\/based rendering. 
\end{DoxyNote}

Constructs a sprite sheet from a texture by dividing it into grid cells. 

Given a texture and frame width/height, this constructor divides the texture into equally-\/sized cells (tiles), and precomputes the number of rows and columns. These are later used to compute UV coordinates per frame.
