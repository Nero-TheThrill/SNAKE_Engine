\doxysubsection{Detailed Description}
Base interface for all renderable/updatable objects. 


\begin{DoxyItemize}
\item Provides lifecycle hooks (Init/\+Late\+Init/\+Update/\+Draw/\+Free/\+Late\+Free).
\item Holds \doxylink{class_transform2_d}{Transform2D}, Material\texorpdfstring{$\ast$}{*}, Mesh\texorpdfstring{$\ast$}{*}, color, tags, and optional components\+: \doxylink{class_sprite_animator}{Sprite\+Animator} and \doxylink{class_collider}{Collider}.
\item Supports render layer tagging and per-\/object visibility/aliveness flags.
\item Supports camera-\/ignored behavior\+: when enabled, world position/scale are derived relative to a reference \doxylink{class_camera2_d}{Camera2D} (see Get\+World\+Position/\+Get\+World\+Scale).
\item Friend with \doxylink{class_frustum_culler}{Frustum\+Culler} to allow visibility tests using \doxylink{class_object_ac5adc371bfc7b4216ffbc85907830e25}{Get\+Bounding\+Radius()}. 
\end{DoxyItemize}
