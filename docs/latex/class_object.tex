\doxysubsection{Detailed Description}
Abstract base class for all renderable and updateable engine objects. 

\doxylink{class_object}{Object} serves as the root class for all drawable, updateable entities within the engine. It provides common interfaces for transform, rendering, collision, animation, and lifetime control.

This class is not meant to be directly inherited or instantiated by users. Instead, users should derive from engine-\/defined subclasses such as \doxylink{class_game_object}{Game\+Object} or \doxylink{class_text_object}{Text\+Object}, which provide concrete implementations and safe access points.

\doxylink{class_object}{Object} enforces lifecycle stages (Init, Update, Draw, Free) and maintains internal flags like is\+Alive, is\+Visible, and ignore\+Camera, while delegating rendering and collision behavior to \doxylink{class_render_manager}{Render\+Manager} and \doxylink{class_object_manager}{Object\+Manager} respectively.

\begin{DoxyNote}{Note}

\begin{DoxyItemize}
\item \doxylink{class_object}{Object} is abstract (contains pure virtual methods).
\item Default constructor is deleted; only protected constructor with \doxylink{_object_8h_a842c5e2e69277690b064bf363c017980}{Object\+Type} allowed.
\item Use Game\+Object/\+Text\+Object as base classes instead.
\end{DoxyItemize}
\end{DoxyNote}
