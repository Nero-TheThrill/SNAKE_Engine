Reports frames-\/per-\/second when an update is due. 

If an FPS update should occur this frame, writes the computed FPS to {\ttfamily out\+FPS} and returns true. Otherwise returns false without modifying {\ttfamily out\+FPS}. The decision is based on the struct\textquotesingle{}s internal accumulators.
