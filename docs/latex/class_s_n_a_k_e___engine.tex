void SNAKE\+\_\+\+Engine\+::\+Run (\begin{DoxyParamCaption}{}{}\end{DoxyParamCaption})}



Runs the main loop until \doxylink{class_s_n_a_k_e___engine_abc2c17ba271c262a62bf2f06ec8334d4}{Request\+Quit()} is called. 

Per-\/frame, processes events, updates input/state/sound, clears the screen, draws the active state, and swaps buffers. Updates the window title with an FPS readout during the loop. On exit, frees Sound/\+State/\+Window and then calls \doxylink{class_s_n_a_k_e___engine_a1d4c98eabd373d6b9cb3e7e5ffa24e3e}{Free()} for final shutdown.
