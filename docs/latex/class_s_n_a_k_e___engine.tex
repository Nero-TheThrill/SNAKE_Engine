\doxysubsection{Detailed Description}
Main entry point and system coordinator for the entire engine lifecycle. 

\doxylink{class_s_n_a_k_e___engine}{SNAKE\+\_\+\+Engine} initializes all subsystems, maintains the main game loop, and provides access to the centralized \doxylink{struct_engine_context}{Engine\+Context}. This class must be created and executed by the user.
