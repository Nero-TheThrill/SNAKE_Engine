\doxysubsection{Detailed Description}
Manages user-\/defined named render layers and maps them to internal layer IDs. 

This class allows users to register named render layers (e.\+g., "{}\+UI"{}, "{}\+Background"{}) and internally maps them to small integer IDs for optimized sorting and rendering.

The system supports up to MAX\+\_\+\+LAYERS layers (default\+: 16), each with a unique name and ID. Layers are used in \doxylink{class_render_manager}{Render\+Manager} to control draw order across multiple categories of objects.

\begin{DoxyNote}{Note}
\doxylink{class_render_layer_manager}{Render\+Layer\+Manager} is used internally by the \doxylink{class_render_manager}{Render\+Manager}. Users register layers through \doxylink{class_render_manager_ae665a48a358691f878caf513fc37e0fa}{Render\+Manager\+::\+Register\+Render\+Layer()}. 
\end{DoxyNote}
