\doxysubsection{Detailed Description}
Handles creation and control of the main GLFW application window. 

Provides window initialization, resizing, title setting, buffer swapping, screen clearing, and event polling. Internally used by \doxylink{class_s_n_a_k_e___engine}{SNAKE\+\_\+\+Engine}.
