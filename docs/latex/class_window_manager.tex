\doxysubsection{Detailed Description}
Manages the GLFW window\+: creation, size/title, clear color, fullscreen, per-\/frame ops. 


\begin{DoxyItemize}
\item Owns the GLFWwindow\texorpdfstring{$\ast$}{*} handle and caches width/height.
\item Initialized via \doxylink{class_window_manager_a2ce03918b6123264cd9013cc4e403f39}{Init()}; sets GL 4.\+6 core profile hints, creates the window, makes the context current, loads GL with glad, enables Open\+GL debug output, installs a debug message callback, and sets the initial viewport.
\item Sets the window user pointer to \doxylink{class_s_n_a_k_e___engine}{SNAKE\+\_\+\+Engine} and registers callbacks\+:
\begin{DoxyItemize}
\item framebuffer\+\_\+size\+\_\+callback\+: calls gl\+Viewport, updates Set\+Width/\+Set\+Height, and forwards the new size to the current state\textquotesingle{}s Camera\+Manager\+::\+Set\+Screen\+Size\+For\+All(width, height) through \doxylink{class_state_manager}{State\+Manager}.
\item glfw\+Set\+Scroll\+Callback/glfw\+Set\+Key\+Callback/glfw\+Set\+Mouse\+Button\+Callback\+: forward to \doxylink{class_input_manager_a9a7e6c988123d556523b698ae010cc5b}{Input\+Manager\+::\+Add\+Scroll} / On\+Key / On\+Mouse\+Button.
\end{DoxyItemize}
\item Typical per-\/frame calls by the engine\+: \doxylink{class_window_manager_aac8852f37e8b30eb90e4ece0279284e6}{Poll\+Events()}, \doxylink{class_window_manager_ac1f44592045972fb10bf4cf991266cb2}{Clear\+Screen()}, \doxylink{class_window_manager_aef8b4e018e5210b06095779adb68203b}{Swap\+Buffers()}. 
\end{DoxyItemize}
