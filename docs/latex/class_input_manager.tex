\doxysubsection{Detailed Description}
Handles per-\/frame keyboard and mouse input polling using GLFW. 

This class manages input state by caching current and previous frames\textquotesingle{} key and mouse button statuses. It supports querying for real-\/time key/mouse down, press, and release events, as well as tracking the cursor position.

\doxylink{class_input_manager}{Input\+Manager} must be initialized with a valid GLFW window before use.
