\doxysubsection{Detailed Description}
Handles real-\/time keyboard and mouse input tracking. 

\doxylink{class_input_manager}{Input\+Manager} tracks both keyboard and mouse input using fixed-\/size bitsets, maintaining the current and previous frame states for all keys and mouse buttons. This allows for precise input queries like Down, Pressed (edge trigger), and Released.

Mouse position is tracked in screen-\/space coordinates, and can also be transformed into world-\/space coordinates using a \doxylink{class_camera2_d}{Camera2D} reference.

\doxylink{class_input_manager}{Input\+Manager} must be updated once per frame using \doxylink{class_input_manager_aa5480931dba2720e7d80dd00a53adae0}{Update()}. \doxylink{class_input_manager_a92319ec0b9f5eff3fd9a2e22a434a3a1}{Reset()} is not called every frame-\/it is only called when the window size changes, such as when the framebuffer is resized.

The system is designed around the GLFW backend and requires a GLFWwindow\texorpdfstring{$\ast$}{*} handle during initialization. \doxylink{class_input_manager}{Input\+Manager} is owned and initialized by \doxylink{class_s_n_a_k_e___engine}{SNAKE\+\_\+\+Engine}.

\begin{DoxyNote}{Note}
This class is not intended to be created or accessed manually by users. 
\end{DoxyNote}

void Input\+Manager\+::\+Update (\begin{DoxyParamCaption}{}{}\end{DoxyParamCaption})\hspace{0.3cm}{\ttfamily [private]}}



Updates the input state by polling GLFW. 

This function should be called once per frame to\+:
\begin{DoxyItemize}
\item Copy current key/mouse state to previous
\item Query latest key/mouse input from GLFW
\item Update current\+Key\+State and current\+Mouse\+State bitsets
\item Retrieve current mouse cursor position
\end{DoxyItemize}

This enables accurate tracking of Pressed/\+Released transitions. Must be called before querying \doxylink{class_input_manager_ac052ac0d0831d40848031aca922a00ef}{Is\+Key\+Pressed()}, \doxylink{class_input_manager_a1adf3a1c58a545d0721479bbbedf81c3}{Is\+Mouse\+Button\+Released()}, etc.

\begin{DoxyNote}{Note}
Internally called by \doxylink{class_s_n_a_k_e___engine}{SNAKE\+\_\+\+Engine} once per frame before logic updates. 
\end{DoxyNote}
