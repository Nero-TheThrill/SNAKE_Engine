\doxysubsection{Detailed Description}
Frame-\/based keyboard/mouse input tracker backed by bitsets. 


\begin{DoxyItemize}
\item Per-\/frame semantics\+: \doxylink{class_input_manager_aa5480931dba2720e7d80dd00a53adae0}{Update()} copies staged-\/\texorpdfstring{$>$}{>}current, current-\/\texorpdfstring{$>$}{>}previous. Press/\+Release queries compare current vs. previous.
\item Mouse\+:
\begin{DoxyItemize}
\item Cursor position is sampled each \doxylink{class_input_manager_aa5480931dba2720e7d80dd00a53adae0}{Update()} via glfw\+Get\+Cursor\+Pos().
\item Scroll\+: \doxylink{class_input_manager_a9a7e6c988123d556523b698ae010cc5b}{Add\+Scroll()} accumulates deltas; \doxylink{class_input_manager_aa5480931dba2720e7d80dd00a53adae0}{Update()} publishes them as per-\/frame scroll\+Delta and resets the accumulator.
\end{DoxyItemize}
\item World-\/space mouse helpers project window coordinates using \doxylink{class_camera2_d}{Camera2D} (position, screen size, zoom). 
\end{DoxyItemize}
