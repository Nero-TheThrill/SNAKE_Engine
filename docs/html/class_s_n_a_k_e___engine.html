Initializes core systems and creates the window.

◆ Run()

void SNAKE_Engine::Run ( )

Runs the main loop until RequestQuit() is called.

Per-frame, processes events, updates input/state/sound, clears the screen, draws the active state, and swaps buffers. Updates the window title with an FPS readout during the loop. On exit, frees Sound/State/Window and then calls Free() for final shutdown.
