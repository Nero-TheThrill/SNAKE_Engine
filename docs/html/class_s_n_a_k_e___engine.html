Detailed Description

Main entry point and system coordinator for the entire engine lifecycle.

SNAKE_Engine initializes all subsystems, maintains the main game loop, and provides access to the centralized EngineContext. This class must be created and executed by the user.

SNAKE_Engine engine;
if (engine.Init(1280, 720))
{
engine.Run();
}
