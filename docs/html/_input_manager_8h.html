enum InputKey

Defines keyboard key codes used by the input system.

These values are directly mapped to GLFW key codes. Use them as parameters in InputManager methods such as IsKeyDown(), IsKeyPressed(), and IsKeyReleased().

if (inputManager->IsKeyPressed(KEY_ESCAPE))
{
engine->RequestQuit();
}

enum InputMouseButton

Defines mouse button codes used by the input system.

These values correspond to GLFW mouse button constants. Use them in InputManager methods such as IsMouseButtonDown(), IsMouseButtonPressed(), and IsMouseButtonReleased().

if (inputManager->IsMouseButtonPressed(MOUSE_BUTTON_LEFT))
{
SelectObject();
}
