Detailed Description

Runtime-configurable texture parameters with sensible defaults.

Defaults:

  • minFilter = Linear
  • magFilter = Linear
  • wrapS = ClampToEdge
  • wrapT = ClampToEdge
  • generateMipmap = true (triggers glGenerateTextureMipmap in GenerateTexture)
