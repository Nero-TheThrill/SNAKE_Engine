◆ FlushDrawCommands()

void RenderManager::FlushDrawCommands ( const EngineContext & engineContext)

Flushes all queued draw batches to the GPU.

  • Iterates layers -> shaders -> (mesh, material, spritesheet) batches.
  • If the first object in a batch is instancing-capable, submits per-instance data; otherwise draws objects one-by-one.
  • Sets view/projection from the batch camera (or identity view); falls back to engine window size when no camera is present.
  • Ensures a bound texture exists; if not, binds the engine error texture.
  • Clears the internal render map afterwards.
