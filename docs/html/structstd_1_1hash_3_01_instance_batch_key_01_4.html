Detailed Description

Custom hash function to support using InstanceBatchKey in unordered_map.

Combines the hash values of the mesh and material pointers. This hash function allows InstanceBatchKey to be used as a key in std::unordered_map.

Note
Uses XOR and left shift to mix two pointer hashes.
