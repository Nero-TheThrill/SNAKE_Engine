Detailed Description

Owns and orchestrates a collection of Object instances.

  • Accepts ownership of new objects, keeps a tag->object map, and exposes utility queries/draw helpers.
  • Provides lifecycle passes (Init/Update/Draw/Free) over managed objects.
  • Offers simple collision checking and collider debug-drawing.
