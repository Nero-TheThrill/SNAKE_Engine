Detailed Description

Bi-directional registry for named render layers.

Maps layer names to 0..MAX_LAYERS-1 IDs and back. MAX_LAYERS is fixed at 16.

bool RenderLayerManager::RegisterLayer ( const std::string & tag,
uint8_t layer )
inlineprivate

Registers a name->ID mapping.

  • Fails (returns false) and logs a warning if the name already exists.
  • Fails (returns false) and logs an error if layer is out of range or the ID is already used.
  • On success, stores both name->ID and ID->name.
Parameters
tagLayer name to register.
layerNumeric ID (0..MAX_LAYERS-1).
Returns
true if registered; false otherwise.
Note
