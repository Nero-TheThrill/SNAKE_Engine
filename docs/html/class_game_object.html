Detailed Description

Minimal Object subclass with no-op lifecycle methods.

  • Constructs the base Object with ObjectType::GAME.
  • All lifecycle hooks (Init/LateInit/Update/Draw/Free/LateFree) are empty by default, so users can override only the ones they need.
