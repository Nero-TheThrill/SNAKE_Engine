Detailed Description

Handles per-frame keyboard and mouse input polling using GLFW.

This class manages input state by caching current and previous frames' key and mouse button statuses. It supports querying for real-time key/mouse down, press, and release events, as well as tracking the cursor position.

InputManager must be initialized with a valid GLFW window before use.

if (engineContext.inputManager->IsKeyPressed(KEY_SPACE))
{
player->Jump();
}
