◆ RenderMap

using RenderMap = std::array<ShaderMap, RenderLayerManager::MAX_LAYERS>

Fixed-size array mapping render layers to ShaderMaps.

RenderMap is a 16-element array (MAX_LAYERS) where each entry represents a render layer. Each layer holds a ShaderMap, which batches objects by Shader and instance data.

Internally used by RenderManager for layered and instanced rendering.

◆ ShaderMap

using ShaderMap = std::map<Shader*, std::map<InstanceBatchKey, std::vector<GameObject*>>>

Render batching structure organized by Shader and instancing key.

Each Shader* points to a map from (Mesh + Material) -> list of GameObjects. Used internally by ObjectManager to prepare instanced draw submissions.
