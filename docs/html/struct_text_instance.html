Detailed Description

Holds font and text content for rendering.

TextInstance stores the font pointer and the actual UTF-8 text string to be displayed. It is used by TextObject to track what text to render and how it should be processed for mesh generation and alignment.

The GetCacheKey() function is used internally to generate a key for caching text meshes.
