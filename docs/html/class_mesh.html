Detailed Description

Represents a GPU mesh composed of vertices and optional indices.

The Mesh class manages vertex and index buffers (VBO, EBO) and their associated VAO. It supports multiple primitive types (e.g., triangles, lines, points), and enables both regular and instanced rendering through OpenGL.

Meshes are constructed from a list of vertices and optional indices. The class also computes a local bounding box for efficient culling or layout.

Note
Mesh instances are usually managed by RenderManager and used by Material.

Constructs a mesh from vertex and optional index data.

Initializes OpenGL buffers (VAO, VBO, and optionally EBO) using the provided vertex data. The mesh can be configured to use triangles, lines, points, etc., through the primitiveType. If index data is not provided, the mesh will be rendered using only vertices.

Parameters
verticesList of vertex data including position and UV.
indicesOptional list of indices. If empty, rendering will be non-indexed.
primitiveTypeThe type of geometric primitive to render (default: Triangles).
std::vector<Vertex> quadVerts = { ... };
std::vector<unsigned> indices = { 0, 1, 2, 2, 3, 0 };
Mesh mesh(quadVerts, indices);
