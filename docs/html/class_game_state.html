Detailed Description

Abstract base class representing a game state or scene.

A GameState represents a self-contained mode in the game such as title screen, gameplay, or pause. It defines a full lifecycle: Load -> Init -> Update/Draw -> Free -> Unload. Each state manages its own ObjectManager and CameraManager.

Override protected methods to implement the specific logic of your game state.
