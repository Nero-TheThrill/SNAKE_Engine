Detailed Description

Base class for a game state (scene/screen).

  • Owns an ObjectManager and a CameraManager.
  • Override the protected virtual hooks (Init/LateInit/Update/LateUpdate/Load/Draw/Free/Unload) to implement state behavior.
  • Internal "System*" methods coordinate the call order and are used by StateManager.
