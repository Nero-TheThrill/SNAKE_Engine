Detailed Description

Represents a GPU-bound material that links a shader, textures, and uniforms.

The Material class encapsulates how an object should appear when rendered. It manages a reference to a Shader, associated textures (by uniform name), and arbitrary uniform values (int, float, vecX, mat4, etc.).

Materials can optionally enable hardware instancing, in which case instance data such as transforms and per-instance uniforms are sent via VBOs.

Note
Materials are usually created and managed by the RenderManager, and assigned to renderable objects such as Mesh or Object.

void Material::SendUniforms ( )
private

Sends all stored uniform values to the GPU via the shader.

Iterates through the internal uniforms map and dispatches each value to the shader using Shader::SendUniform(), based on its type (int, float, vec2, etc.).

This function is automatically called during Bind() and is not intended to be invoked directly by users.

Note
If a uniform name is invalid or not used by the shader, the shader may silently ignore it.

void Material::SetTexture ( const std::string & uniformName,
Texture * texture )
inline

Associates a texture with a uniform sampler name.

Stores a pointer to the given Texture under the specified uniform name (e.g., "u_MainTex"). This mapping is maintained in the internal texture map and used when the material is bound. Textures are automatically assigned to texture units in order during Bind().

This function does not immediately bind the texture to the GPU; it only records the mapping for later use.

Parameters
uniformNameThe name of the sampler2D uniform in the shader.
texturePointer to the Texture object to associate.

◆ SetUniform()

void Material::SetUniform ( const std::string & name,
UniformValue value )
inline

Stores a uniform value for later use during rendering.

Adds or updates a uniform value in the internal uniforms map, indexed by the uniform's name. The value is stored as a std::variant that supports int, float, vec2, vec3, vec4, and mat4.

Uniforms are not sent immediately to the GPU. Instead, they are batched and submitted all at once in SendUniforms() during the material Bind() call.

Parameters
nameThe name of the uniform variable in the shader.
valueThe value to associate with the uniform (int, float, vec2, etc.).
material.SetUniform("u_Tint", glm::vec4(1.0f, 0, 0, 1.0f));
