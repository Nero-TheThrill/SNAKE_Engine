Detailed Description

Represents a GPU-bound material used to render objects.

A Material encapsulates a Shader, a set of bound textures, and uniform values. It also optionally supports GPU instancing using a per-instance matrix buffer.

  • Textures are set using SetTexture().
  • Uniforms (int, float, vectors, matrices) are set using SetUniform().
  • Call EnableInstancing() to enable per-instance rendering.

Used internally by ObjectManager and RenderManager to prepare draw submissions.

◆ SetTexture()

void Material::SetTexture ( const std::string & uniformName,
Texture * texture )
inline

Binds a texture to a named uniform slot in the shader.

This method registers a texture to be used for a given uniform name. The texture is not immediately bound to the GPU; instead, it is recorded internally and later sent to the shader via SendUniforms().

The uniform name must match a sampler2D (or similar) declared in the shader.

Parameters
uniformNameThe name of the sampler uniform in the shader.
texturePointer to the Texture to be associated with this uniform.
material->SetTexture("u_DiffuseMap", texture);

void Material::SetUniform ( const std::string & name,
UniformValue value )
inline

Stores a uniform value to be sent to the shader during rendering.

This method does not immediately send the value to the GPU. Instead, it stores the value in an internal map, and the uniform is sent later via SendUniforms(), typically right before drawing.

Supported types include:

  • int
  • float
  • glm::vec2
  • glm::vec3
  • glm::vec4
  • glm::mat4
Parameters
nameName of the uniform variable in the shader.
