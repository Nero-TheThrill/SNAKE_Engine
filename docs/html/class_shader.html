Detailed Description

Represents an OpenGL shader program composed of multiple shader stages.

The Shader class handles loading, compiling, linking, and using GPU shader programs. It supports uniform uploading and optional instancing checks. Shaders can be built from external source files, one per stage.

Typical usage includes loading a shader, linking, and then binding it before rendering.

◆ Link()

void Shader::Link ( )
private

Links all attached shaders into a single GPU program.

Validates presence of paired tessellation shaders (TCS + TES) if either is used. Reports and logs OpenGL linking errors. Also checks whether the shader supports instancing via a specific attribute ("i_Model").

After linking, all shaders are detached but not deleted.
