Detailed Description

Tracks frame timing and calculates delta time and FPS.

Provides per-frame timing data such as delta time and frame rate updates. Intended to be called once per frame from the game loop.

Usage:

EngineTimer timer;
timer.Start();
while (running)
{
float dt = timer.Tick();
if (timer.ShouldUpdateFPS(fps))
{
std::cout << "FPS: " << fps << std::endl;
}
}
