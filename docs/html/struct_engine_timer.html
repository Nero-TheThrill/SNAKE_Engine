Detailed Description

Utility class for tracking delta time and calculating FPS.

EngineTimer provides timing utilities for the engine's main loop. It measures time between frames (delta time), tracks frame count, and determines when to update FPS metrics (typically every 0.3 seconds).

The timer should be started once at the beginning of the game loop using Start(), and Tick() should be called once per frame to get delta time.
