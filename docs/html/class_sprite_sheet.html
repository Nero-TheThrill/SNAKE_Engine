Detailed Description

Grid-based sprite sheet built from a texture and fixed frame size.

  • The sheet is partitioned into (columns x rows) using frameWidth/Height. Texture size is read from Texture::GetWidth()/GetHeight().
  • UV helpers expose per-frame offset and scale for a grid cell.
  • Clips (name -> SpriteClip) can be registered and queried.
Note
Texture* is non-owning; the texture must outlive the sheet.
