Filters visible objects using 2D frustum culling.

Iterates through a list of objects and determines whether each object is within the visible bounds of the given camera and viewport. Objects flagged as "ignore camera" are always included. The result is stored in the outVisibleList.

Parameters
cameraThe 2D camera used to define the view bounds.
allObjectsThe list of all objects to be tested.
outVisibleListOutput list containing only visible objects.
viewportSizeSize of the rendering area in screen pixels.
Note
This is an internal helper used by RenderManager for visibility culling.
